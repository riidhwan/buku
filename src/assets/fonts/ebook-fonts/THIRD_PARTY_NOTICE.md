# ebook-fonts Core Collection

Buku bundles the Core Collection from `other-core-fonts.zip` in the ebook-fonts release
`v2026.07.02`.

Source: https://github.com/nicoverbruggen/ebook-fonts/releases/download/v2026.07.02/other-core-fonts.zip

The ebook-fonts README describes the collection as free/libre fonts modified for e-reading.
Most fonts are available under the Open Font License, with font-specific attribution and
copyright messages preserved in the font files and documented by the upstream project.

Upstream project: https://github.com/nicoverbruggen/ebook-fonts
